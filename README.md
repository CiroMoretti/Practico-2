# **Resolución Práctico 2**:
## Composición de la entrega:
    - "src": Módulos de Python.
    - "Notebooks": Archivo Jupyter Notebook para ejecutar los Módulos.

## Requisitos:
    - Python 3.13.12 en adelante.
    - Jupyter Notebook, o su extensión en VSCode (la cual fue usada para la resolución del práctico).

## Instalación:
    - Descargar/Clonar el repositorio en su máquina.
    - Abrir el archivo contenido en /Notebooks.
    - Ejecutar las celdas. Los códigos son importados desde la carpeta /src.
